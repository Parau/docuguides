Assuma o papel de um autor de material didático. Você deverá gerar um texto que explica o que é o prompt no contexto do ChatGPT para um leigo em tecnologia. Você deverá usar a seguinte estratégia em sua explicação. Abrir o texto explicando de forma mais geral o que é um prompt evitando o uso de termos técnicos. Na sequencia você abrirá um segundo tópico de texto que, baseado no texto de abertura vai aprofundar um pouco mais os conhecimentos sobre prompt, novamente deve evitar o uso de termos técnicos. por fim você concluirá a explicação de prompt em um terceiro tópico que, baseado no que foi abordado no tópico anterior, aprofunda ainda mais os conhecimentos sobre o prompt e agora pode lançar mão da terminologia técnica, mas sempre explicando o que é o termo e, quando necessário, usando analogias. O ultimo tópico do texto você vai de forma objetiva falar sobre como é essencial desenvolver a habilidade de formular prompt bem estruturados — uma competência conhecida como engenharia de prompts. São os bons prompts que conseguem extrair o máximo dessas ferramentas poderosas.
